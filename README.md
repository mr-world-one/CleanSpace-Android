# CleanSpace 🧹

**CleanSpace** — Android-додаток для очищення пам'яті телефону від непотрібних файлів, кешу та дублікатів.

---

## 📋 Про проект

**Проблема:** Android-смартфони накопичують гігабайти "сміття", яке важко знайти вручну.

**Рішення:** CleanSpace автоматично сканує сховище та безпечно видаляє непотрібні файли.

---

## 👥 Для кого

- **Всі, у кого забивається телефон**

---

## ⚙️ Функціонал

- 🔍 Сканування сховища
- 🗂️ Пошук кешу, тимчасових файлів, дублікатів
- ✅ Безпечне видалення
- 📊 Статистика очищень
- 🛡️ "Білий список" для важливих папок

---

## 🛠 Технології

- **Мова:** Java
- **Android:** API 23+ (Android 6.0)
- **База даних:** SQLite/Room
- **UI:** XML, Material Design
- **Архітектура:** MVP
